-----

勤快的朋友可以全部扫进去，方便切换着用，嫌麻烦只需要扫五六个即可。

**MAC电脑**：点击上面的链接即可打开二维码，然后小飞机下拉菜单 - 服务器 - 从屏幕上扫描二维码

**苹果手机**：点击上面的链接即可打开二维码，然后 shadowrocket 里左上角按钮点一下即可进入扫码界面，扫上面的二维码即可

**安卓手机**：点击上面的链接即可打开二维码，然后 小飞机客户端左上角，三角形点进去，右下角+号，选 扫描二维码
